Este archivo busca enumerar los archivos entregados en este proyecto y explicar la función y contenido de cada uno.

NOTEBOOKS:
----------
-Notebook "ProyectoDataScienceUM.ipynb": es el único notebook que se utilizó para el proyecto. En el mismo hay código
así como celdas Markdown que van explicando todo lo que se hizo y por qué. También incluye las problemáticas que se
fueron analizando y conclusiones finales.

DATASETS:
----------
-"Indicators.csv" es el dataset principal utilizado en el trabajo. Contiene los datos por país y por año de más de
mil indicadores. Fuente: World Bank.
-"Indicator_selection.csv" es un dataset que contiene los indicadores seleccionados para armar el modelo en la etapa
de 'Modeling' del trabajo. Fuente: World Bank.

ARCHIVOS DE TEXTO:
----------
-"README.txt": contiene información de todos los archivos entregados.
